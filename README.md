## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Upgrade pip
* pip
  ```sh
  pip install --upgrade pip
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/NicoRio42/athlete-website.git
   ```
2. pip
  ```sh
  pip install requirements.txt
  ```
